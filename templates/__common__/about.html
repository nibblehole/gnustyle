About this searx instance

What is searx?

Searx is a metasearch engine, aggregating the results of other search engines while not storing information about its users.

What does searx.gnu.style do different from a vanilla searx?

searx.gnu.style focuses on privacy, speed and security.

Privacy

  • All webserver logs are sent to /dev/null. This increases privacy as well as speed. However if you try to hack the instance with for instance sql injection or XSS queries, your actions will be logged.

Speed

  • To achieve speed the search query response is capped at 0.6 seconds for the general category. If you experience lack of search results you should try submitting the query again.
  • searx.gnu.style strives for the first places on searx.space. Besides that the instance is regularly checked with Google PageSpeed Insights, Pingdom's Website Speed Test and GTmetrix Website Speed Analysis where it gets scores of 100 or close to 100. If you would like to see the test results, click on the following links: Google PageSpeed Insights , Pingdom Speed Test, GTmetrix Analysis

Security

  • searx.gnu.style enforces HTTPS connections and HTTP/2 transfers. Test results: SSL Labs Report, Security Headers Report, CryptCheck, Immuniweb, HSTS Preload, Mozilla TLS Observatory
  • Content Security Policy is implemented at the webserver level to protect against XSS, click-jacking and other attacks. If you would like to check the test results, click on the following links: Google CSP Evaluator, Mozilla HTTP Observatory CSP Validator
